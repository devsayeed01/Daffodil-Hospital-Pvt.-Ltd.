# Daffodil-Hospital-Pvt.-Ltd.
A Hospital Management System webpage. Project for CSE Database Course.
The healthcare industry is increasingly reliant on digital management systems to ensure accuracy and efficiency in patient care. The Daffodil Hospital Pvt Ltd. is a web-based application designed to bridge the communication gap between patients, medical staff, and administrators. Currently, many healthcare facilities rely on manual record-keeping or fragmented systems that lead to scheduling conflicts and delayed emergency responses. This project aims to solve the problem of disorganized patient scheduling and resource management by providing a centralized database for doctors, diagnostic tests, and ambulance services.

Objectives -> To design a relational database that securely stores patient, doctor, and staff credentials.

-> To implement a multi-role authentication system for Admins, Doctors, and Patients.

-> To create a seamless appointment booking interface that links patients directly with specialized doctors.

-> To manage a 24/7 emergency ambulance fleet and a catalog of pathology services through a unified dashboard.

-> To ensure data reliability through the use of MySQL transactions and rollback mechanisms.

Existing hospital management systems often range from simple scheduling apps to enterprise-level software like Oracle Health. While commercial systems offer high complexity, they are often too expensive for smaller private hospitals. This project contributes a lightweight, cost-effective web application using the PHP/MySQL stack. Similar to existing platforms, this system supports CRUD (Create, Read, Update, Delete) operations for hospital resources but specifically focuses on the immediate needs of a localized facility like Daffodil Smart City and the Savar area.
